Sidenotes Demo

The sidenotes plugin converts standard Markdown footnotes into inline margin notes placed next to their reference in the text.1 It requires no jQuery and works with any Markdown processor that outputs the standard footnote HTML structure.

When two footnotes appear in the same paragraph they are automatically grouped together.2 This keeps the margin tidy and prevents overlap.3 The group is treated as a single unit for placement purposes.

The plugin exposes a small API: show(), hide(), placement(), and destroy(). Use the buttons below to try them out.4

  1. Sidenotes are a typographic convention popularised by Edward Tufte. They keep supplementary content visible without interrupting reading flow. ↩
  2. Grouping logic: two sidenotes share a group when their reference marks are both direct children of the same block element (their common pivot). ↩
  3. The CSS for positioning is entirely your responsibility. The plugin only manages DOM structure; the visual margin offset is achieved with position: absolute; left: 100% on the sidenote element. ↩
  4. The full API: new Sidenotes(element, options). See the README for the complete options reference. ↩
